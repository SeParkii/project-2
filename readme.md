# Concert Journal

## Overview

Concert Memory Journal is a CRUD-based web application where users can record the concerts they have attended.
The website allows users to store key details, such as artist name, concert name, venue, city, date, ticket type, ticket price, and seat information, and revisit these memories anytime.

# Features

1. Add New Concert Entry
   Users can fill out a form to save details of a concert they attended.

2. View All Concert Memories
   The homepage displays all stored ticket records in a clean card layout.

3. Edit Concert Information
   Users can update details at any time through the Edit function.

4. Delete a Concert Entry
   Users can remove outdated or incorrect data using the Delete button.
